mips、arm 环境部署及调试

.

环境搭建：https://reverseengineering.stackexchange.com/questions/8829/cross-debugging-for-arm-mips-elf-with-qemu-toolchain
各种库都可以用apt来安装
查看被占用的端口ID:
sudo lsof -i:port
kill掉该进程
sudo kill -9 850
调试
先在qemu下打开
qemu-aarch64(architecture type) -g port -L /usr/aarch64-linux-gnu/(lib directory) ./pwn
socat 搭建
socat tcp-l:10002,fork exec:”qemu-aarch64 -g 1234 -L /usr/aarch64-linux-gnu ./pwn”,reuseaddr
然后用gdb-multiarch调试
set architecture aarch64 (设置汇编)
target remote localhost:1234
就能够进行调试

如果直接运行报错

设置好环境变量就能够运行了
参考：https://xz.aliyun.com/t/3154
